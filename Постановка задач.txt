Задача 1

Название задачи: "Создание макета страниц"
Описание задачи: Нарисовать макеты стартовой страницы сайта и страницы авторизации для десктопной и мобильной версий. Создать иконки для уроков.
Направление: UX/UI

Задача 2

Название задачи: "Реализация формы регистрации"
Описание задачи: Создать форму регистрации через email/телефон с вводом пароля и подтверждением контакта. Добавить экран выбора роли (Студент/Преподаватель, по умолчанию Студент).
Направление: Frontend / Backend

Задача 3

Название задачи: "Интеграция SSO (Google/Microsoft/Apple)"
Описание задачи: Реализовать регистрацию через SSO с созданием связанной локальной учётной записи и запросом недостающих полей (имя, согласия).
Направление: Frontend / Backend

Задача 4

Название задачи: "Логирование согласий"
Описание задачи: Добавить согласие пользователя с офертой и политикой конфиденциальности, логировать согласия с фиксацией времени и версии документа.
Направление: Backend

Задача 5

Название задачи: "Валидация идентификаторов"
Описание задачи: Реализовать проверку формата email/телефон на клиенте и сервере.
Направление: Frontend / Backend

Задача 6

Название задачи: "Реализация входа с паролем и MFA"
Описание задачи: Реализовать вход по паролю с опциональной MFA (TOTP/SMS/email-код). Добавить проверку прав на уровне UI и API.
Направление: Frontend / Backend

Задача 7

Название задачи: "Реализация passwordless входа"
Описание задачи: Добавить вход по магической ссылке/email-коду как фичу-флаг.
Направление: Frontend / Backend

Задача 8

Название задачи: "Ограничение попыток входа и капча"
Описание задачи: Ограничить число попыток входа, добавить капчу после N неудачных попыток.
Направление: Backend / Security

Задача 9

Название задачи: "Управление MFA в кабинете"
Описание задачи: Реализовать добавление/удаление факторов MFA в личном кабинете с подтверждением пользователя.
Направление: Frontend / Backend / Security

Задача 10

Название задачи: "Выход из системы"
Описание задачи: Реализовать локальный (текущая сессия) и глобальный (все устройства) выход.
Направление: Frontend / Backend / Security

Задача 11

Название задачи: "Восстановление доступа"
Описание задачи: Реализовать сброс пароля через подтверждённый email/телефон с одноразовой ссылкой/кодом и TTL. Добавить проверку второго фактора при включенной MFA.
Направление: Frontend / Backend / Security

Задача 12

Название задачи: "Управление сессиями"
Описание задачи: Показать список активных сессий в кабинете пользователя и возможность завершить любую сессию.
Направление: Frontend / Backend / Security

Задача 13

Название задачи: "Внедрение RBAC"
Описание задачи: Реализовать роли student, teacher, admin с гранулярными разрешениями (просмотр/редактирование курсов, управление группами, модерация). Реализовать делегирование прав преподавателем.
Направление: Backend / Security

Задача 14

Название задачи: "Локализация интерфейса"
Описание задачи: Сделать интерфейсы входа/восстановления доступными на RU/EN языках, расширяемо.
Направление: Frontend / UX

Задача 15

Название задачи: "Доступность клавиатурой"
Описание задачи: Обеспечить поддержку фокуса клавиатурой на всех элементах.
Направление: Frontend / UX

Задача 16

Название задачи: "Безопасное хранение паролей"
Описание задачи: Хранить пароли в зашифрованном виде (bcrypt/Argon2), проверять сложность по политике (минимум 10 символов, классы символов).
Направление: Backend / Security

Задача 17

Название задачи: "Защита от брутфорса"
Описание задачи: Реализовать защиту от брутфорса (капча или временная блокировка).

Задача 18

Задача 19

Задача 20

Название задачи: "Юзабилити сообщений об ошибках"
Описание задачи: Сделать сообщения об ошибках чёткими, без раскрытия, какой параметр неверный.
Направление: Frontend / UX
